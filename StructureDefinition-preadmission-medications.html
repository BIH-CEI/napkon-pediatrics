Resource Profile: Preadmission Medications

Defining URL:https://www.napkon.de/fhir/StructureDefinition/preadmission-medications
Version:1.0.0
Name:PreadmissionMedications
Title:Preadmission Medications
Status:Active as of 2021-12-03
Definition:

List of medications taken before admission to a hospital

Publisher:Charité - Universitätsmedizin Berlin
Source Resource:XML / JSON / Turtle

The official URL for this profile is: 

https://www.napkon.de/fhir/StructureDefinition/preadmission-medications

Guidance

This List profile is used to collect all medications that were taken prior to medication, as defined in the Immunosuppresive Permanent Medication profile (all entries must be derived from that profile).

Parameter Case Report Form FHIR Resource Attribute
Which immunosuppressive/-modulatory medications did the patient take before admissions? List.entry
