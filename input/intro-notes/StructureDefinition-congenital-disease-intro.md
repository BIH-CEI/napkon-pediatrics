### Guidance

| Parameter Case Report Form | FHIR Resource Attribute |
| -------------------------- | ----------------------- |
| Does the patient suffer from a congenital disease? Which congenital disease does the patient suffer from? | `Condition.code` |
{:.grid}

The `snomed` slice of the `Condition.code` is restricted to a value set that contains all children of `#66091009 "Congenital disease (disorder)"`. The specific congenital disease that a patient suffers from can be specified using any of the respective child codes.
